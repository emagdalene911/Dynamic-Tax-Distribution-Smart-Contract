# 🏛 Dynamic Tax Distribution Smart Contract

A cutting-edge smart contract built on Stacks blockchain that brings tax collection and distribution into the Web3 era! 

## 🚀 Overview

This smart contract revolutionizes how governments can handle tax collection and distribution in a transparent, efficient and automated way.

## ✨ Features

### 💰 Tax Collection
- Automatic tax calculation based on wallet balance
- Current tax rate: 10% (configurable)
- Seamless STX token transfers to government treasury

### 👑 Government Controls  
- Secure government address management
- Treasury balance monitoring
- Department-wise fund allocation

### 📊 Public Transparency
- Check individual tax payments
- View department allocations
- Monitor treasury balance

## 🛠 Functions

### Public Functions
- `pay-tax`: Pay your taxes automatically based on wallet balance
- `set-government-address`: Update the government wallet address
- `allocate-funds`: Distribute funds to different departments

### Read-Only Functions
- `get-tax-payment`: Check how much tax someone paid
- `get-department-allocation`: View funds allocated to a department
- `get-treasury-balance`: Check current treasury balance

## 🔐 Security Features
- Authorization checks for government functions
- Amount validation for all transfers
- Safe arithmetic operations
